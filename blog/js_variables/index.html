JS變數的性質

By Reid Chu 06 June 2021


變數值

  1. 如果變數沒有被指定任何值，為 undefined

    var notDesignated;

if (notDesignated === undefined){
    console.log("變數未指定特定值");
};

  2. undefined 對應 Boolean 為 false

  3. 被賦予 undefined 的變數與數值運算後回傳 NaN

  4. 如果 null 與數值運算，null 會被視為 0，且 null 之 Boolean 為 false

動態型別

JavaScript 是動態型別的語言，在宣告變數時並不需要指定型別，會依據輸入值而自動改變。因此當 String 與 Number 同時操作時容易混淆。

  1. 同時使用 Number 與 String 時，JavaScript 會將 Number 視為 String

    x = "The answer is " + 42
console.log(x)  // "The answer is 42

  2. 其他運算子例如減號，就不會將 Number 視為 String，也就是可以直接運算。但相反的，使用加號的話，卻會被轉換為字串

    "108" - 8 //100
"108" + 8 //1088

  3. 綜合以上兩點，依據不同情況需指定變數的型別，避免混淆

    • parseInt()
    • parseFloat()
