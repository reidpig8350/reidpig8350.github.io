軟體工程師扶持計畫訓練營

By Reid Chu 20 January 2021

個人簡介

在求學時對於自然科學比較有興趣的原因，是因為我喜歡在知道答案之後，梳理每一個步驟的感覺。除此之外，我也喜歡團隊合作的討論與交流，除了可以從旁人的經驗獲取知識，也能夠透過講解的過程更加深自己的能力，屢試不爽。因此在職涯的抉擇中，就在學術研究以及工程師這兩個領域之中作出抉擇，但苦於對機械的興趣無法再更深，覺得機械領域並不是適合我的選擇，也在寫程式的過程中獲得成就感，才會有想要成為工程師的念頭。

我畢業於中山大學機械系，在畢業之際也汲汲營營和系上同學一起報考研究所，幸運錄取了台大應力所。
碩班一年的修課期間，在「計算材料力學」的課程中，教授以 Python 為計算工具，讓我們得已將過去人腦運算的公式，轉譯成電腦的語言（例如人腦可以運算無限，電腦則否），除了學期末的作業讓我獲得滿滿的成就感，在每個星期與隊友的討論也讓我學習到不少，是我第一次接觸程式。

然而後來論文難有進展，而對學術研究這條路懷疑，因此給自己轉彎的機會，申請休學後就開始尋找工作。
目前在網頁數據分析的公司擔任PM，雖然應徵條件上並沒有程式語言的要求，但是卻提供了我一些作為程式語言練習的素材。例如客戶週報與月報會需要 Google Analytics(GA) 所搜集的資料，以及後續的資料整理使用的 Excel，我分別利用 Python 定時抓取 Google Analytics api 資料，再使用 VBA 整理 Excel 中存取的數據，除了能夠大幅的減少我工作的時間，也降低我粗心的機會。

為什麼想成為工程師

世界上有許多大受歡迎的產品，都是從一個微小的念頭開始的，在生活中常常都會有一閃而過的點子，但是都苦於紙上談兵，難有實踐的機會。例如我曾經在報章上，閱讀到日本的高中生，利用「搜集發票並且結合消費者資訊」，成功開啟自己的創業生涯。就是這樣簡單一句話帶過的點子，卻能夠改變一個人的人生。

因此我想要成為工程師的重要原因，是不想要再錯過。

我認為想要讓大家發現自己點子最好的方法，不是真的本錢超級多，就是真的把東西給造出來。因此希望能從成為工程師作為我前進的很難跨越、很艱辛也很巨大的一步。

我希望我可以是一位專精於前端，仍了解後端的工程師。因為根據過去的學習經驗，我適合藉由專精的過程觸類旁通。 除此之外，跨領域的議題一直都會吸引我的注意，前端包含了使用者的使用體驗以及行銷策略等等，我認為是工程與其他領域的交會處，也是我想要了解的原因之一。

為了成為工程師，曾經做過什麼努力？

  1. 彭彭老師的 Python 與 JS 課程。
  2. 目前的工作崗位並沒有程式語言的要求，但是為了讓自己練習各式工具，也把許多公司內工程師的任務納入自己的守備範圍。例如以 Python 抓取 api 的資料、為客戶的網頁追蹤資料（例如：點擊「加入會員」按鈕的次數）、客戶的Google Tag Manager建立與撰寫VBA製作月報等等，盡可能讓自己「熟悉不熟悉的事物」。
  3. 正在學習的 Udemy 課程，包含了前後端工程師的基礎，目前學習了HTML、JS，以及編輯網頁常用的jQuery及Bootstrap等等函式庫以及工具。
  4. 每天下班之後最少仍要求自己接觸程式語言，或者網頁的相關知識
  5. 目前正在起步階段的專案，利用 Python 撰寫自己的投資理專
  6. 自己假扮多人修改程式碼，練習 git 工具

對於工程師的職業想像是什麼？

工程師是必須長時間與電腦為伍的職業，有許多問題必須要獨力解決。正面的意義是具備足夠的能力，然而負面的角度則是較缺乏整體性的考量。例如 UI 或者 UX，或者是和 PM 之間的認知的差異等等。

我認為工程師是十足的勞工，因為問題的存在只能是 0 或者 1，如果沒有克服困難，那麼產品永遠不能上線。因此如果沒有時間管理或者是工作效率的技巧，就必須要耗費大量的時間或者金錢成本以完成專案。

就積極面而言，我認為工程師是「可以」甚至是「必須」不斷學習的身份。和經驗累積的做中學不同，工程師有超前部署學習的責任，也就是讓自己的技術超越現在的位置。這對我而言也許是挑戰，但是也是會讓我想要成為工程師的原因之一。

如果參與這個計畫，會怎麼安排學習時間？

經過一年的工作，我越來越確定並不會將這條路當成職涯的規劃，因此領完年終後就要離開現任崗位，繼續學習工程師的技能。

因此若有幸參與訓練營，大約只會有第一個月與目前工作的重疊時間，且暫時不會尋找新的工作。

因此預計會協助家裡的事業，全職學習成為工程師。

如果參與這個計畫，預期會碰到什麼困難？你打算怎麼解決它？

首要問題是經濟問題，目前大約足夠我六個月的生活開銷，若有不足家人也願意體諒。除此之外，有在接案網站上嘗試 VBA 的案子，是工作之外的額外收入。

以及生涯規劃的問題是，目前仍留有台大應力所的學籍，雖然是休學狀態，但休學的期限只到明年的二月。因此在為期六個月的訓練營中，也要為明年的復學做準備。

想要對我們說的事情？

謝謝彭彭老師在 Youtube 的課程，雖然內容是針對入門學習，但是每次回去複習的時候總是會有收穫。

我對於實體的活動非常期待，因為我很喜歡和大家一起合作以及討論的感覺，雖然這次的身份是作為學生，也許沒有辦法輸出技術，但是能夠和經驗豐富的前輩與會，一定會收穫良多。
